# Pablo's Claude Code Plugins

A marketplace of plugins for Claude Code.

## Installation

Add this marketplace to Claude Code:

```
/plugins marketplace add https://github.com/Pgarciapg/pgarciapg-plugins
```

Then browse and install plugins with `/plugins`.

## Available Plugins

### design-team
A multi-agent design department for Claude Code: UX, visual, copy, accessibility, tokens, and implementation.

**Commands:**
- `/kickoff` - Start a design project
- `/critique` - Get design feedback
- `/spec` - Generate design specs
- `/tokens` - Create design tokens

### domain-scout
AI-powered domain name finder that analyzes your codebase, generates creative name suggestions, and searches registrars for availability and pricing.

**Commands:**
- `/domain-scout` - Find domain names for your project

**Features:**
- Analyzes your codebase to understand the project
- Generates 5 brandable domain name suggestions
- Searches Namecheap, Porkbun, GoDaddy, Squarespace Domains, and Cloudflare
- Compares prices across registrars
- Requires Chrome + Claude in Chrome extension
